# CSV Export

Download your cached Strava activities as a CSV file.

## How to Export

1. Go to **Strava → Activities** in the WordPress admin
2. Click the **Export CSV** button (appears when activities are cached)
3. A file named `strava-activities-YYYY-MM-DD.csv` downloads to your browser

## Exported Columns

| Column | Description |
|---|---|
| Title | Activity name |
| Type | Ride, Run, Walk, etc. |
| Distance | Distance in your configured unit |
| Unit | mi or km |
| Duration | Formatted moving time |
| Date | ISO 8601 date |
| Elevation Gain | Metres |
| Avg Speed | In mph or km/h |
| Max Speed | In mph or km/h |
| Speed Unit | mph or km/h |
| Avg HR | Average heart rate (if available) |
| Max HR | Maximum heart rate (if available) |
| Calories | Estimated calories |
| Kudos | Kudos count |
| Comments | Comment count |
| City | Starting city |
| Country | Starting country |
| Private | Yes or No |
| Strava URL | Link to activity on Strava |

## Use Cases

- Backup your activity data
- Import into spreadsheets for analysis
- Share with coaches or training partners
- Data migration between sites
